P2P Messaging Bootstrap Server

This is a bootstrap server for the P2P messaging application. It helps peers discover each other on different networks.

API Usage

To register a peer and get a list of other peers, send a POST request to /api/bootstrap with the following JSON data:

{
  "user_id": "unique_user_id",
  "username": "your_username",
  "ip": "your_public_ip",
  "port": your_public_port,
  "pubkey": "your_public_key_base64"
}

The server will respond with a list of registered peers that are currently online.

Status

The bootstrap server is currently running and available for peer registration.
